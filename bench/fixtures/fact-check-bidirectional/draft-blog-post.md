# Draft: "What's New in NimbusSync v4.2"

_Draft for the release blog post — please fact-check against the source
docs before this goes out._

NimbusSync's v4.2 release brings a handful of changes we think teams will
notice right away.

**Smarter conflict handling.** NimbusSync now supports automatic conflict
resolution for simultaneous edits, so two people editing the same document
offline no longer means someone's changes get silently overwritten when
they reconnect.

**Encryption for shared folders.** NimbusSync added end-to-end encryption
for shared folders in the latest release, giving teams an extra layer of
protection on folders shared outside the immediate team.

**Offline mobile editing.** The mobile app now supports offline editing
with automatic sync on reconnect — edit a file on a flight, and it syncs
the moment you're back online.

**Native Microsoft Teams integration.** NimbusSync now integrates natively
with Microsoft Teams for real-time notifications, so your team sees sync
activity right inside the channels they already use.

**Faster sync.** NimbusSync reduced average sync latency by 40% in the
latest release, thanks to a rewritten diffing engine.

**Unlimited storage on the free tier.** Great news for anyone on the free
plan: NimbusSync now offers unlimited storage for all free-tier accounts,
no more worrying about hitting a cap.

We're excited for teams to try all of this out — full release notes are
linked below.
