# Task: production work order for a launch-promo image edit

Halfmoon Labs (makers of the Lumen Notes app) has a launch-promo image
change due tomorrow. The request came in over a client/PM thread, and our
production team has a few different tools that could be used to execute
it.

Attached in this directory:

- `request/client-thread.md` — the client/PM thread describing what they
  want changed, the constraints, and the deadline.
- `request/tooling-inventory.md` — the production tools (and people)
  currently available to execute this with.
- `assets/hero-spec.md` — a description of the current hero illustration
  that the change applies to. (No image file is attached — this is a
  planning step, not an image-editing step. Treat the spec as ground
  truth for what the artwork looks like.)

Please write `work-order.md` for whoever executes this. Cover:

1. Which route/tool you're choosing, and why.
2. Exact steps (or exact prompt text, if the chosen tool takes one)
   specific enough that someone else could execute it without having to
   guess anything.
3. How the result will be checked before it ships.

Constraints: this is a planning/decision step — don't attempt to produce
an actual image file. Don't access anything outside this directory.
